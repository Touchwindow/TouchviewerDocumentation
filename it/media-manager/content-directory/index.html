    Elenco Contenuti

    Cartella Archivio

    Qui è possibile vedere a quale archivio si sta attualmente puntando.
    Cliccando su questa icona è possibile cambiare archivio, puntando ad un altra cartella.
    Cliccando su questa icona si effettua un refresh dei contenuti e cartelle dell'archivio
    Cliccando su questa icona verrà aperta la cartella dell'archivio in Esplora Risorse.

    Cartella

    Viene mostrata la struttura delle cartelle e dei file dell'archivio a cui si sta puntando.
    Si può inoltre modificare l'archivio, aggiungendo o rimuovendo cartelle e file in esso contenuti.
    Facendo click col tasto destro del mouse sulle cartelle o sui contenuti si aprirà un sottomenu da cui è poissibile effettuare diverse operazione descritte qua di seguito.

    Sottomenu cartella

    1. Aggiorna la cartella
    2. Apre la cartella in Esplora Risorse
    3. Elimina la cartella e tutti i suoi contenuti
    4. Si accede alla Opzioni della cartella. Vai alla sezione Opzioni Cartella
    5. Aggiunge un contenuto alla cartella. Vai alla sezione Aggiungi Contenuti
    6. Crea una sottocartella
    7. Nasconde la cartella corrente e noin viene mostrata nel Media Presenter
    8. Si può collegare questa cartella ad un altra. L'effetto è che nel Media Presenter cliccando questa sezione verranno visualizzati i contenuti della cartella linkata.

    Sottomenu file

    1. Apre il file con il programma di default di windows assegnato per quella tipologia di file
    2. Elimina il file dalla cartella
    3. Si accede alla Opzioni del File Vai alla sezione Opzioni Contenuto
    4. Il file viene duplicato
