    REQUISITI

    Hardware

    Requisiti minimi
    Processore: I3 di seconda generazione
    Scheda video: HD2000, Nvidia / Ati 500Mb di memoria
    Ram: 1GB

    Requisiti consigliati
    Processore: i3 di terza generazione o superiore
    Scheda video: HD4000, Nvidia / Ati 1GB di memoria
    Ram: 2GB

    Software

    Sistemi Operativi
    Windows 7
    Windows 8.1
    Windows 10 Anniversary Edition

    Librerie
    Microsoft .NET Framework 4.5 (Installato automaticamente come prerequisito)
    K-lite Standard edition (Installato automaticamente come prerequisito)
    VLC (Per abilitare funzionalità streaming video - Scarica VLC)
