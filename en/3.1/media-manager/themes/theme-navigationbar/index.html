    Navigation bar theme

    The navigation bar can also be customised for each individual section, setting the following options:

    Navigation Type

    Sets the type of bar in 2 different modes:

    • Bar complete with navigation and function button: the navigation bar will be complete with "Breadcrumbs" navigation and function buttons.
    • Function buttons only: only function buttons will be shown.

    Bar position

    • Top: the bar will be positioned at the top of the screen.
    • Bottom: the bar will be positioned at the bottom of the screen.

    Closing [Only in complete bar mode]

    • Always visible: the navigation bar always stays visible.
    • Allow closing: a button to show or hide the navigation bar will be shown.

    Outer margin colour [Only in complete bar mode]

    Sets the navigation bar container margin colour.

    Margin thickness [Only in complete bar mode]

    Sets the navigation bar container margin thickness.

    Background colour [Only in complete bar mode]

    Sets the navigation bar background colour.

    Shade

    Sets the shade that the navigation bar should have. You can choose from:

    • None
    • Low
    • Medium
    • High

    Folder/button colour [Only in complete bar mode]

    Sets the colour of folders and function buttons.

    Folder font [Only in complete bar mode]

    Sets folder font.

    Current folder font [Only in complete bar mode]

    Sets current folder font.

    Button type [Only in button function only mode]

    • Standard: the buttons displayed will be those standard loaded during installation.
    • Custom: Custom buttons can be loaded. Buttons must have already been loaded in their desired colour.

    Button colour [Only in standard button function only mode]

    Sets the colour of standard buttons.
